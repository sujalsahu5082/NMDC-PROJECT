# Contributing to NMDC-PROJECT

Thank you for your interest in contributing! 🎉

## How to Contribute

### 1. Fork & Clone
```bash
git clone https://github.com/YOUR-USERNAME/NMDC-PROJECT.git
cd NMDC-PROJECT
```

### 2. Create a Branch
```bash
git checkout -b feature/your-feature-name
```

### 3. Make Your Changes
- Follow existing code style
- Keep commits small and focused
- Write meaningful commit messages

### 4. Test Your Changes
- Verify the login page works
- Check that the backend API responds correctly
- Test with sample Excel data

### 5. Submit a Pull Request
- Open a PR against the `main` branch
- Describe what you changed and why
- Link any related issues

## Code Style

- **HTML/CSS/JS**: Use 2-space indentation
- **Python**: Follow PEP 8 guidelines
- Use descriptive variable and function names

## Reporting Bugs

Open an [Issue](https://github.com/sujalsahu5082/NMDC-PROJECT/issues) with:
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable

## Questions?

Open a GitHub Discussion or reach out via Issues.
